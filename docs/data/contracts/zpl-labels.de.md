# Spezifikation: ZPL-Etiketten

Definiert zulässige Platzhalter und die Formatierung für Zebra-Druckeretiketten.

| Platzhalter    | Quelle             | Beispiel         |
| -------------- | ------------------ | ---------------- |
| `{{BATCH_ID}}` | Prüflos            | `2025-09-04-001` |
| `{{USER}}`     | Aktueller Bediener | `pruefer_a`      |
